Hypothesis: Elk Hunting has increased in Montana due to out of state interest.
Hunting by Montana Residency

Time series and bar graph data from: https://myfwp.mt.gov/fwpPub/harvestReports

*To access the same data that was used in this study, select the following parameters from the Harvest Estimates Search Menu:
Species: Elk
Start Year: 2004
End Year: 2020
District: All

The number of elk hunting days indicate the total number of days that elk were collectivley hunted
by all hunters in the state of Montana in a given year. This is further broken up by Montana residency status to determine
if out-of-state hunters are contributing to increased elk hunting days in Montana. Residents are defined as those
who legally call Montana home. Non-residents are those who legally reside in a different state..
