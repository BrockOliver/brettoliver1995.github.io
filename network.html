Used the Scholarly python module to search for all papers on Google Scholar with the keywords "Geodetic water storage", which returned 28 papers.
Scholarly sorts data into tuples of coauthor pairs with the following format: [('author 1', 'author 2), ('author 1, author 3', 'author 2, author 2', et..)],
Used pythonic dictionaries to convert data from scholarly into a JSON-readable type (shown in the bottom block of code). "dd" is the final dictionary used for visualization.
